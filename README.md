# 汽车俱乐部

## 项目概述

本项目服务于厦门线下实体汽车俱乐部（名字待定），在服务好俱乐部业务的同时进一步探索汽车类互联网产品创新模式。

## 项目载体

项目最终以微信小程序为呈现载体，为了方便读者理解微信小程序与 APP 的区别，简单做了一个对比表格如下：

> 注意：下表中的好坏高低都只是二者相对而言。

| | 微信小程序 | APP  | 说明 |  
| ---  |  ----  | ----  | --- | 
| 研发成本 | 低  | 高 | 因为 APP 需要 IOS 与 Android 两个平台开发，虽然有 Flutter 之类的宣称跨平台的框架，但是相对而言 APP 开发量还是较大 |
| 上线速度 | 快  | 慢 | 同上 | 
| 消息推送 | 差  | 好 | 小程序受微信限制，不如 APP  | 
| 获客成本 | 低  | 高 | 小程序便于通过微信进行推广，用户点开即用，不需要下载安装  | 
| 其他因素 | 多  | 少 | 小程序界面的每一次更新都需要通过微信审核，所涉及的经营业务也会要求上传相关资质 | 

## 项目模块

项目分为以下几个模块：

- 首页

首页以图文形式提供资讯类服务，包括：新车介绍、汽车改装、驾驶体验等。

- 车友圈

该板块供注册用户进行交流讨论用，实际上就是个论坛。发帖子的形式有：图文、视频。后期用户基数做大后，可以考虑做延伸的功能，例如寻找同城同类车友等。

- 选车

该板块提供新车、二手车的销售服务。提供通过品牌、价格、销量等维度进行筛选的功能。
用户选中车子后提交订单（填写个人联系方式即可，无需付款，因为汽车款项较大，不适合通过微信付款），由俱乐部工作人员后续跟进进行详细沟通。

- 我

该板块为用户的个人中心版块。可以查看个人收藏的帖子、发布的帖子、提交的订单、站内消息、关注、粉丝等。
